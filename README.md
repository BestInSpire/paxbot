# paxbot
Paxanya sunucusu için özel yapılmış discord botu.

# Requirements
<ul>
  <li><a href="https://nodejs.org/en/">Node.JS v16</a></li>
</ul>
# Setup
<ul>
  <li>Download Project</li>
  <li>Configure .env file</li>
  <li>$ npm install -g yarn</li>
  <li>$ yarn install</li>
  <li>$ yarn dev</li>
</ul>
